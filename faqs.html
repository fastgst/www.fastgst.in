Find answers to common questions about our GST Tax Lookup API, pricing, and integration.

FastGST.in is a platform that provides GST Tax Lookup API services, offering access to accurate GST tax rates, HSN/SAC codes, and classification data for Indian businesses and developers. We're operated by Sangeeth Nandakumar, an independent developer building GST-compliant solutions.

Currently, we offer:

  • GST Tax Lookup API with HSN/SAC code search
  • Real-time tax rate information
  • Fuzzy search capabilities
  • Hierarchical classification data
  • Cess details and conditional tax scenarios

We're also developing a full GST-compliant invoicing platform that will be available soon.

Our data is sourced from official CBIC and GST Council notifications and is updated regularly to ensure accuracy and compliance with current tax regulations. We strive to stay in sync with the latest government notifications to keep our data fresh and relevant.

Integration is straightforward:

  1. Sign up at developers.fastgst.in
  2. Get your API key from the dashboard
  3. Make HTTP requests to our endpoints
  4. Receive JSON responses with tax information

Check out our documentation for detailed integration guides and examples.

Our API is RESTful and can be used with any programming language that supports HTTP requests. We provide examples in popular languages like JavaScript, Python, PHP, Java, and more in our documentation. The API returns standard JSON responses that are easy to parse in any language.
